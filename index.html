Skills

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar o conteúdo de um documento eletrônico, como uma página da web.
CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência de um documento escrito em HTML, como fontes, cores e layout.
Bootstrap é um conjunto de ferramentas de código aberto para desenvolvimento de projetos de front-end (parte visível do site) com HTML, CSS e JavaScript. Ele fornece um conjunto de componentes pré-estilizados que podem ser facilmente incorporados em um projeto, economizando tempo e esforço de design.
JavaScript é uma linguagem de programação de alto nível interpretada, amplamente utilizada na web para criar efeitos interativos em páginas da web.
jQuery é uma biblioteca JavaScript que facilita o uso do JavaScript em projetos web, oferecendo uma sintaxe simplificada e recursos úteis para manipulação do DOM (Document Object Model), animação, requisições AJAX e muito mais.
PHP (PHP: Hypertext Preprocessor) é uma linguagem de programação de script criada originalmente para o desenvolvimento de aplicações web. Ele é executado no lado do servidor e pode ser usado para criar páginas da web dinâmicas e acessar bancos de dados.
Laravel é um framework de aplicação PHP que fornece uma série de ferramentas e recursos para ajudar os desenvolvedores a criar aplicações web de alta qualidade de maneira rápida e eficiente. Ele inclui recursos como roteamento, gerenciamento de dependências, gerenciamento de banco de dados e autenticação de usuário.
